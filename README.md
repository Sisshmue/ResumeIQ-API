# ResumeIQ API

ResumeIQ API is an AI-powered resume analysis service that evaluates resumes, provides ATS-style feedback, and extracts important keywords to help job seekers improve their resumes.

The API allows users to upload a resume and receive structured analysis and suggestions generated by AI.

## Features

* AI-powered resume analysis
* ATS-style feedback and improvement suggestions
* Resume keyword extraction
* Resume-to-job-description matching
* File upload support (PDF resumes)
* API documentation with Swagger
* Rate limiting for API protection
* Cloud deployment

## Tech Stack

* Node.js
* Express.js
* Multer (file uploads)
* Swagger UI
* AI API integration
* Render (cloud deployment)

## Live API

Base URL:

https://resumeiq-api.onrender.com

API documentation:

https://resumeiq-api.onrender.com/api-docs

## API Endpoints

### Analyze Resume

Upload a resume and receive AI feedback.

POST `/api/analyze/resume`

Form Data:

resume (PDF file)

Example request:

curl -X POST https://resumeiq-api.onrender.com/api/analyze/resume 
-H "Content-Type: multipart/form-data" 
-F "resume=@resume.pdf"

Response:

{
"message": "Resume uploaded successfully",
"analysis": "...AI feedback...",
"keywords": ["javascript", "nodejs", "react"]
}

### Match Resume With Job Description

Upload a resume and provide a job description to receive match analysis.

POST `/api/match/resume`

Form Data:

resume (PDF file)
jobDescription (text)

## Installation

Clone the repository:

git clone https://github.com/Sisshmue/ResumeIQ-API.git

Navigate to the project folder:

cd ResumeIQ-API

Install dependencies:

npm install

## Environment Variables

Create a `.env` file and add your API keys:

PORT=3000
AI_API_KEY=your_api_key_here

## Run the Server

Development mode:

npm run dev

Production mode:

npm start

Server will run at:

http://localhost:3000

## Deployment

The API is deployed on Render and supports automatic deployment from GitHub.
